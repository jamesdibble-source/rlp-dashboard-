### 12.5 RLP Lot Size Adjustment Table (THE key analytical tool)
Maps % change in lot size → % adjustment in price:

| Lot Size Discount | Price Adjustment |
|---|---|
| 0-10% | -2.24% |
| 10-20% | -5.40% |
| 20-30% | -12.61% |
| 30-40% | -17.37% |
| 40-50% | -24.39% |
| 50-60% | -38.66% |
| 60-70% | -47.50% |

**Key insight:** Price doesn't move linearly with lot size. A 30% smaller lot is only 17% cheaper. This means $/m² INCREASES as lots get smaller — critical for understanding density economics.

### 12.6 RLP & OPC Prorating Tool
- Input: Default lot size, RLP, desired lot size → Outputs adjusted RLP
- Input: Default lot size, OPC, desired lot size → Outputs adjusted OPC
- Uses the adjustment table above
- Example: 500m² at $250K → 600m² = $281,533 (non-linear)
